# IBM Lookup File VSCode Support

## Introduction
Netcool/Tivoli Lookup Language (.lookup) is a proprietary language to write lookups for IBM Tivoli Netcool/Objectserver Probes. 

## Features
- Syntax Highlight for .lookup Files
- more to come :-)
